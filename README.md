# University Form Website

A simple University Registration Form website built using HTML and CSS.
This project is designed to collect student details through a clean and user-friendly form interface.

## Features

* Student registration form
* Clean UI design
* Responsive layout
* Navigation bar
* Input fields for student details
* Styled using CSS

## Technologies Used

* HTML5
* CSS3

## Project Structure

```bash
University-Form/
│── index.html
│── style.css
│── About.html
│── Contact.html
│── Registration_form.html
│── Courses_page.html
```

## How to Run the Project

1. Download or clone the project
2. Open the project folder
3. Run `index.html` in your browser

## Purpose of the Project

This project was created for practicing frontend web development concepts like:

* HTML forms
* CSS styling
* Navigation bars
* Page linking
* Responsive layouts
